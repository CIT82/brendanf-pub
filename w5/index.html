Radio Buttons



How would I use Radio Buttons in my Webpage?


I would use ratios in a form for a multiple choice selection of different answers to choose from. Only one answer can be selected at a time unlike checks that allow you to pick more than answer. If you pick a different answer then it will change your selection to that new answer. I used radios in this webpage for a selection of possible choices of different pets people might have. The downside to using radios for this is that you cannot pick more than one pet. The first option is check by default, which will be selected by default when the page when the page loads. The last option is disabled, so you cannot pick it (None of you are James Bond villians). But enjoy the Gif for the last option.

The bootstrap styles used for radios on this page have changed some defaults styles used by the browser for input and label elements. There are some other bootstrap default styles available, but are being used in this form. The radio buttons change with the icon style and color used. The font for the text under the label elements change as well. I am a little disappointed that I am not able to change the flex styling and the box model sytling to make the form look nicer. Either the bootstrap css extensions are not working or the bootstrap css variables are not available to use for these input and label element examples provided by them. I was able to change the opacity for the last options, (form-check-input-disabled-opacity) and (form-check-label-disabled-opacity) by disabling the opacity styles used for the disabled ratio input buttons.
